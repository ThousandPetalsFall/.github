# ThousandPetalsFall

> 每一片花瓣都能绽放光芒。

我们把日常生活里的小需求，做成轻量、好用、可分享的产品。

## 我们在做什么

- `轻工具`：把高频但零散的小需求做成马上能用的工具
- `小程序`：围绕微信生态和多端体验做轻量产品
- `内容型产品`：把记录、分享、思考变成可持续的产品体验
- `AI 应用`：把智能问答、内容生成和自动化能力融入真实场景

## 项目地图

### 🤝 社交与陪伴

- **partnerhub** - 找搭子平台，覆盖通勤、游戏、活动、饭搭子和旅行伙伴等场景
- **treehole** - 匿名树洞社交，给情绪一个安全出口
- **checkin-circle** - 面向多人共同打卡的习惯养成社区
- **follow-im** - AI 客服系统，支持多客服、多用户和知识库优化

### 🌱 习惯与生活

- **daily-checkin** - 每日打卡与习惯培养工具
- **today-plan** - 今日计划管理与公开分享
- **todo-list** - 简洁的待办事项管理与分享功能
- **mood-diary** - 每日心情记录与公开社区分享
- **emotion-analysis** - 通过对话帮助用户理解情绪背后的原因
- **water-reminder** - 喝水提醒与健康饮水习惯培养

### ✈️ 出行与效率

- **packing-list** - 旅行行李清单管理和协作
- **travel-footprint** - 记录旅行足迹、地图展示、相册管理和团队协作
- **currency-exchange** - 实时汇率查询与换算
- **aa-accounting** - 朋友聚餐、旅行等场景的 AA 制记账分摊工具
- **photo-tools** - 证件照与图片处理工具合集

### 📚 记录与表达

- **item-storage** - 拍照记录物品位置，支持分类管理和分享
- **book-notes** - 记录读过的书和感想，支持社区分享
- **urban-notes** - 面向城市、资源与社会问题的深度思考文章集合

### 🔮 灵感与探索

- **astrology** - 星座运势与 MBTI 内容工具
- **cyrus-jesus** - 基于 AI 的圣经经文推荐与灵修内容生成
- **tarot** - 塔罗牌占卜小程序
- **draw-lots** - 轻量抽签与决策辅助工具

### 🧩 基础能力

- **resource-hub** - 资源共享小程序，支持微信小程序端
- **unified-backend** - ThousandPetalsFall 组织内项目统一后端 API 服务

## 技术栈

- 前端：uni-app + Vue 3
- 后端：FastAPI + Python
- 数据库：SQLite / MySQL
- AI：MiniMax API

## 组织定位

- 以小程序、轻应用和内容产品为主
- 倾向做能快速验证、容易传播、适合持续迭代的项目
- 统一后端服务逐步为多个项目提供公共能力支撑
